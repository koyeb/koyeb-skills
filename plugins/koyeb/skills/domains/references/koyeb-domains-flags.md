# Domains Command Flags Reference

Complete reference for all `koyeb domains` command flags.

## List/Get/Describe
```
koyeb domains list [flags]
koyeb domains get NAME [flags]
koyeb domains describe NAME [flags]
```
No command-specific flags.

## Create
```
koyeb domains create NAME [flags]
  --attach-to string   Attach to app on creation
```

## Attach/Detach
```
koyeb domains attach NAME APP [flags]
koyeb domains detach NAME [flags]
```
No command-specific flags.

## Refresh
```
koyeb domains refresh NAME [flags]
```
No command-specific flags.

## Delete
```
koyeb domains delete [flags]
```
No command-specific flags.
