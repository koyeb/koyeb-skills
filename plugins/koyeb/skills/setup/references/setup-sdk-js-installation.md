# Koyeb Sandbox SDK (JavaScript/TypeScript) Installation

## Install

### npm

```bash
npm install @koyeb/sandbox-sdk
```

### pnpm

```bash
pnpm add @koyeb/sandbox-sdk
```

### yarn

```bash
yarn add @koyeb/sandbox-sdk
```

## Verify installation

```bash
npm list @koyeb/sandbox-sdk
```

Should output the installed version.

## Test import

```javascript
import { Sandbox } from '@koyeb/sandbox-sdk';
console.log('SDK loaded successfully');
```
