# Koyeb Skills

These skills provide step-by-step guidance for managing Koyeb resources with the Koyeb CLI and the Koyeb Sandbox SDKs.

## Requirements

### CLI skills
- Koyeb CLI installed and available on PATH
- Authenticated session (`koyeb login`) or `KOYEB_API_TOKEN` set
- Network access to Koyeb APIs

### Sandbox SDK skills
- API token available as `KOYEB_API_TOKEN`
- Network access to Koyeb APIs
- JavaScript/TypeScript: Node.js with `@koyeb/sandbox-sdk` installed
- Python: Python with `koyeb-sdk` installed

## Notes

Skills are structured for progressive disclosure. Each skill includes a minimal, focused instruction set and references for detailed command usage.
