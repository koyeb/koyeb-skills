# Koyeb Skills

These skills provide step-by-step guidance for managing Koyeb resources with the Koyeb CLI.

## Requirements

- Koyeb CLI installed and available on PATH
- Authenticated session (`koyeb login`) or `KOYEB_TOKEN` set
- Network access to Koyeb APIs

## Notes

Skills are structured for progressive disclosure. Each skill includes a minimal, focused instruction set and references for detailed command usage.
