# Claude Code Prompt Pack

Use these prompts one-by-one in a fresh Claude Code session.

1. "List all Koyeb apps in my org."
2. "Create a new app named `demo-app` and then create a service for it."
3. "Pause the app `demo-app` and confirm it is paused."
4. "Show logs for service `demo-service`."
5. "Deploy the latest build for `demo-service`."
6. "Attach the domain `example.com` to service `demo-service`."
7. "Create a secret called `DATABASE_URL` with value `postgres://...` and then reveal it."
8. "List deployments for `demo-service` and show the latest deployment logs."
9. "List instances for `demo-service` and exec `ls -la` in the first instance."
10. "Create a volume named `data-vol` and list volumes."
11. "Create a database named `demo-db` and list databases."
12. "What version of the Koyeb CLI is installed?"

If any step is destructive for your account, modify names or skip as needed.
